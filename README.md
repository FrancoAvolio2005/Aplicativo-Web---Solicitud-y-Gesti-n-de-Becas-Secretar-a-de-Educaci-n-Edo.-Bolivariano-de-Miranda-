# Solicitud y Gestión de Becas
 Aplicación Web para manejar la Solicitud y Gestión de Becas en la Secretaría de Educación del Edo. Miranda. Cuyo sistema busca automatizar los procesos que se llevan acabo dentro del ente educativo. Metodología de trabajo (SCRUM).

# Detalles del Aplicativo Web
 Para el desarrollo del proyecto se utilizaron las siguientes herramientas (lenguajes, frameworks y librerías) como: <b> HTML, Python (Django), JavaScript, SQLite, Bootstraps, Chart.Js, Openpyxl y Jazzmin.</b>

 La aplicación se sustenta bajo los siguientes patrones de diseño como: <b> Model - view - template (MVT), Principio de Responsabilidad Única (SRP), Principio de Segregación de Interfaces (ISP), Command (Command Pattern) y Strategy (Strategy Pattern).</b>
